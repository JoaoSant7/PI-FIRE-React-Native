# PI-FIRE-React-Native

Este é o front end de nosso Projeto Integrador do terceiro período do Curso de Análise e Desenvolvimento de Sistemas da Faculdade SENAC.

## Ferramentas utilizadas

Para desenvolver este projeto nós utilizamos uma série de ferramentas, sendo elas:
- React Native (framework)
- Javascript (linguagem)
- Yarn (package manager)
- Expo GO (tester)
